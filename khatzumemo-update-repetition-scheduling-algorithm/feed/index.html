#1) I made the scheduling stricter so that you have to really LOOK at the item and THINK about how you’re going to remember it somewhat, and essentially earn yourself a 5-point score (maybe more than one)…that should only happen with new items or items with a bad track record. I was wanting this to make you really think “are ya sure you know this? really?”…

If you find that too strict I might change it, or make it so that that item comes up later on that same day.
